# Agent Cockpit v0.4.15

## Shipped For Users

- Fixed Windows installer reruns so reinstalling or repairing Agent Cockpit no longer fails when the existing private Node.js runtime is still in use.

## Developer Details

- Updated `scripts/install-windows.ps1` to reuse an existing verified private Node.js runtime instead of deleting the target runtime directory.
- Added a fallback repair path that installs a fresh sibling runtime directory when the expected runtime path exists but cannot be verified.
- Stopped the existing PM2 app on a best-effort basis before replacing an existing Windows release app directory.
- Added static regression coverage and updated deployment/testing specs for repair-friendly Windows reinstall behavior.
