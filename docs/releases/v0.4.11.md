# Agent Cockpit v0.4.11

## Shipped For Users

- Fixed Windows installs so Agent Cockpit stays in the background instead of leaving a visible Node console window on the desktop.
- Windows logon startup now launches the Agent Cockpit start helper in a hidden PowerShell window while still using the current user's scheduled task.

## Developer Details

- Added `windowsHide: true` to the PM2 ecosystem config generated by `scripts/install-windows.ps1` and the Windows production self-update path in `src/services/updateService.ts`.
- Added `-WindowStyle Hidden` to the Windows Scheduled Task PowerShell action and to detached updater restart launches.
- Added focused regression coverage in `test/windowsInstallerScript.test.ts` and `test/updateService.test.ts` for hidden Windows runtime startup.
- Updated README, deployment specs, and backend service specs to document the hidden Windows PM2 and scheduled-task behavior.
