# Agent Cockpit v0.4.16

## Shipped For Users

- Fixed Windows installer reruns when the existing private Node.js folder is incomplete or partially deleted.

## Developer Details

- Tightened `scripts/install-windows.ps1` private-runtime reuse checks to require npm's JavaScript entrypoints before accepting an existing runtime directory.
- Added explicit native command exit-code handling for Node/npm version probes so failed `.cmd` invocations cannot be treated as successful verification.
- Kept the repair path that installs a fresh sibling runtime directory instead of deleting a locked or corrupt runtime folder.
- Updated static installer coverage and deployment/testing specs for corrupt-runtime repair behavior.
